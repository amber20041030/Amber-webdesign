# 自訂網站編輯器

這版修正「雙擊文字不能編輯」和「滑鼠移上去一直變成移動鍵」的問題。

## v11 重點

- 元件內容區不再是移動游標
- 只有元件邊框上方跳出的「移動」把手可以拖曳
- 雙擊文字可直接進入編輯模式
- 編輯文字時會隱藏移動把手、工具列、縮放點
- 區塊仍然照 HTML 順序排列，區塊本身不能自由拖移
- 區塊內的元件仍可自由排版、拖移、縮放

## 操作方式

1. 先新增 Header / Section / Footer 等區塊
2. 點選區塊
3. 新增文字、標題、圖片、按鈕等自由元件
4. 單擊元件：選取
5. 拖曳元件邊框上方「移動」把手：移動
6. 拖曳藍色控制點：縮放
7. 雙擊文字：編輯
8. Esc：結束文字編輯 / 取消選取
9. Delete：刪除
10. Ctrl+C / Ctrl+V：複製貼上

## v84 修正

### GitHub 連結 / 其他電腦存檔失敗
