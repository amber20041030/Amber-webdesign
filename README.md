# bootstrap-v95-image-pure-zoom-preview-fix

本版以 V94 為基礎修正：

1. 圖片滑鼠移入放大改為「純放大」
   - 不再額外套用亮度、陰影、透明、混合模式等效果。
   - 放大時固定以圖片中心點放大。
   - 原本圖片裁切、圓角、尺寸設定仍保留。
   - 若圖片原本有黑白設定，放大時不會再被切成其他顏色效果。

2. 預覽模式底部不再容易被吃掉
   - 預覽時會保留底部安全空間。
   - 右下角「返回編輯」按鈕不會蓋住最底部內容。
   - 預覽畫面高度會重新計算，避免最後一段內容被視窗或捲動區裁切。

3. 保留 V94 既有功能
   - 圖片圖層可正常修改。
   - 圖片上方有其他圖層遮住時，滑鼠移到圖片範圍仍會觸發放大。
   - 匯出 HTML ZIP 後仍保留圖片純放大效果。

檔案：
- `index.html`：多檔版入口
- `css/editor.css`：樣式
- `js/editor.js`：主要功能
- `bootstrap-v95-單檔版.html`：單檔版
