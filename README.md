# v137 匯出與預覽一致修正版

本版接續 v136。

## 修改重點

1. 匯出網站會自動套用與預覽模式相同的顯示狀態。
2. 匯出 `css/style.css` 會收集目前編輯器實際載入的 CSS 與動態 style，避免新功能在預覽正常、匯出後缺樣式。
3. 匯出 HTML 的 `<body>` 會同時帶有 `exported-site preview-mode`，讓所有預覽模式 CSS 可直接生效。
4. 匯出 JS 補強通用互動初始化：
   - 圖片 hover 放大 / 黑白效果
   - 下拉式選項開合與選項點擊
   - 導覽列下拉選單
   - 頁面跳轉、錨點滾動、連結
   - YouTube iframe 顯示
   - 保留原有輪播、下拉換組、手風琴、側邊欄、hover slide 等匯出互動
5. 不修改原本資料結構、不影響手動存檔歷史紀錄、不新增清空快捷鍵。

## 使用方式

使用 `index.html` 開啟編輯器。完成設計後按「匯出 ZIP」，解壓縮後開啟匯出的 `index.html`，顯示與預覽模式一致。
