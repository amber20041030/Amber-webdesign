## v134 修正

- 工具列顯示改由選取/啟用模板狀態控制，不再受滑鼠移入、移出影響。
- 點擊其他元件、其他模板或畫布空白處才取消目前下拉式換組模板工具列顯示。
- 模板內元件可自由拖移到模板原本範圍外，不再強制限制 left/top 邊界。

- 原本上一組、下一組、複製組、刪除組、複製內容、貼上內容與新增組邏輯保留。


## v134 修正
- 下拉式換組模板的區塊 / 模板邊界改回只計算模板本體，不把浮動工具列算入範圍。
- 移動工具列與組別工具列維持浮動顯示，不佔用模板內部空間。
- 下拉式換組控制下拉元件改以 `.select-switcher-block` 模板本體作為拖移定位範圍，不再被外層區塊邊界誤限制。
- 控制下拉元件拖移後會限制在模板本體內，避免跑出模板；其餘換組、複製、刪除、貼上邏輯不變。
