# v157 - 形狀元件不滿版延伸修正

本版基於 v154 修改：

1. 修正預覽 / 匯出後左右仍可能殘留細縫的問題。
2. 區塊背景滿版寬度增加安全補邊，避免不同電腦、瀏覽器縮放或小數點計算造成背景沒占滿。
3. 新增「區塊內滿版背景元件也延伸到視窗」設定。
4. 新增「此元件預覽 / 匯出時滿版延伸」設定，僅適合圖片背景；形狀元件不再滿版延伸。
5. 內容元件仍維持固定畫布基準，不會因不同電腦視窗寬度跑版。
6. 保留 v154 全部功能。


## v156
- 滿版背景左右安全外擴加大到 240px，避免不同電腦/縮放比例造成左右露底。
- 形狀/圖片滿版背景元件在預覽與匯出時使用固定畫布縮放基準重新計算寬度。
- 同步補強 body、canvas-area、區塊 pseudo 背景，避免左側殘留未滿版區域。


## v157
- 形狀元件不再參與滿版延伸，避免換電腦後形狀被拉成整個視窗寬度。
- 滿版背景仍保留在頁面 / 區塊底，圖片背景元件仍可滿版。
- 保留 v156 的左右安全外擴與固定畫布縮放。
