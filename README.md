# v144-multiselect-animation-setting

本版延續 v143 的「元件多選後可一次更改設定」，新增動畫面板多選同步。

## 修改重點

- 多選元件後，可一次套用「基礎滑動出入動畫」設定。
- 支援動畫啟用、方向、距離、初始 X/Y、滑入 X/Y、動畫時間。
- 多選時只同步目前正在變動的動畫欄位，不會覆蓋位置、大小、文字、背景或其他樣式。
- 方向與距離會依原本邏輯同步更新初始位移，維持原動畫操作方式。
- 保留單選、區塊動畫、預覽與匯出播放邏輯。

## 檔案

- `index.html`：一般專案版
- `js/editor.js`：主要程式，已加入 v144 補丁
- `v144-單檔版.html`：單檔版
