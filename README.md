# v141 - 多選單欄位套用與下拉換組文字保存修正

本版以 v140 為基礎，只修正以下問題，其餘邏輯不改變：

1. 多選一次調整設定時，只套用目前變動的欄位。
   - 修改文字顏色不會連帶改變字級、位置、圓角、圖層。
   - 修改背景顏色不會連帶覆蓋透明度或材質。
   - 修改透明度不會連帶覆蓋原本顏色。
   - 修改圓角 / 字級 / 行高 / 字距 / 圖層時，只改該項設定。

2. 修正下拉式換組模板內下拉元件的選項文字。
   - 選項文字會保存到下拉換組模板資料。
   - 修改其他元件後不會跳回預設「選項 1 / 選項 2 / 選項 3」。
   - 上一組 / 下一組切換後文字仍保留。
   - 重新整理、手動存檔、歷史紀錄、匯出後仍保留。

3. 保留原本 v140 功能。
   - 模板內反藍多選保留。
   - 下拉換組功能保留。
   - 匯出與預覽一致修正保留。
   - 低階桌機效能模式保留。
   - Ctrl + P 歷史紀錄保留。
   - 沒有新增任何清空快捷鍵。


## v142 新增：複製分頁

- 新增「複製分頁」按鈕。
- 會先保存目前分頁的最新畫面，再複製成一個新分頁。
- 複製內容包含目前分頁的元件、區塊、樣式、背景、各裝置模式資料與互動設定。
- 複製後會自動切換到新分頁。
- 不改變原本新增、改名、刪除分頁功能。
- 不影響下拉換組、多選、歷史紀錄、匯出與預覽邏輯。

## v143 修改內容
- 元件反藍多選後，右側設定面板可一次套用到所有已選取元件。
- 調整某一個設定時，只更新該欄位，不會連帶覆蓋其他位置、大小或樣式。
- 支援一般樣式、文字樣式、背景、材質、位置單欄位、陰影、形狀、線條、圖片圓角/裁切/放大效果等多選同步。
