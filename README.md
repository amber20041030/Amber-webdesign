# v139 低階桌機效能版 + 歷史紀錄快捷鍵

基於 v138 修正，新增以下調整：

1. 依使用者電腦規格調整成低效能模式：Intel Pentium Gold G6405、8GB RAM、64 位元桌機、無觸控螢幕。
2. 編輯器預設加入 `low-spec-editor` / `no-touch-device`，降低背景網格、陰影、動畫、過多 transition 對 CPU/GPU 的負擔。
3. 自動存檔節流：低效能模式下自動存檔延後為約 4.5 秒，MutationObserver 變更也會合併處理，減少拖移與大量編輯時卡頓。
4. 新增歷史紀錄快捷鍵：`Ctrl + P` 開啟「歷史存檔紀錄」，並阻止瀏覽器列印視窗彈出。
5. 歷史紀錄功能維持 v136 邏輯：每次手動存檔建立 1 筆、顯示時間與版本、可還原、可刪除、最多保留 5 筆。
6. 沒有新增任何清空快捷鍵；清空仍只保留原本按鈕操作。
7. 保留 v138 匯出修正：匯出後元件不消失，保留預覽實際 DOM。

更新檔案：

- `index.html`
- `css/editor.css`
- `js/editor.js`
- `v139-單檔版.html`
